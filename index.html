Chicken Run Instant Game Privacy

Chicken Run Instant Game重视用户的隐私。您在使用我们的服务时，我们可能会收集和使用您的相关信息。我们希望通过本《隐私政策》向您说明，在使用我们的服务时，我们如何收集、使用、储存和分享这些信息，以及我们为您提供的访问、更新、控制和保护这些信息的方式。本《隐私政策》与您所使用的小游戏服务息息相关，希望您仔细阅读，在需要时，按照本《隐私政策》的指引，作出您认为适当的选择。本《隐私政策》中涉及的相关技术词汇，我们尽量以简明扼要的表述，并提供进一步说明的链接，以便您的理解。 Chicken Run Instant Game会通过facebook提供的官方api对用户数据进行存储和读取，用户数据只局限于跟游戏状态相关的数据，不涉及个人信息。跟您相关的用户名、头像等信息我们会通过facebook提供的官方api来获取，并承诺只使用于游戏界面展示，不会存储到第三发服务器。 我们可能使用在您的游戏过程中，向您提供与您相关的广告。广告提供来自facebook平台，不包含第三方广告平台。 我们可能适时修订本《隐私政策》的条款，该等修订构成本《隐私政策》的一部分。
